QUICK START

Running in 5 Minutes

1
Install Python 3.10+

Download from python.org. Then open a terminal and run:
pip install discord.py groq openai huggingface_hub

2
Create Your Discord Application

Go to discord.com/developers/applications → New Application → Bot tab → Reset Token. Copy your token.

3
Enable Required Intents

In the Bot tab: scroll to Privileged Gateway Intents and enable both Message Content Intent and Server Members Intent.

4
Get a Free Groq API Key

Sign up at console.groq.com (no credit card). Create an API key and copy it.

5
Fill the Generator & Download

Paste your keys into the generator above, customize your character, then click Generate & Download.

6
Run & Invite

Open terminal in the folder with your bot.py and run python bot.py. Invite the bot to your server from the OAuth2 tab in the Developer Portal (select bot scope + permissions).

7
Activate in a Channel

In your Discord server, run /activate #your-channel. Then mention the bot or reply to start chatting!

Frequently Asked Questions

Is my Discord token safe?+
Completely. Your token never leaves your browser. The generator runs 100% client-side in JavaScript — our servers never see your credentials. The bot.py file is built locally and downloaded directly to your device.
Do I need coding knowledge?+
No coding required. Just fill in the form, download the file, open a terminal and run python bot.py. Basic terminal usage is all you need.
What is Groq and why is it free?+
Groq is an AI inference provider with extremely fast LLaMA model access. They offer a generous free tier at console.groq.com — no credit card required. Your bot uses it to power all AI character responses.
Why isn't my bot responding?+
Check these in order: (1) Run /activate #your-channel in your Discord server. (2) Make sure Message Content Intent and Server Members Intent are enabled in the Discord Developer Portal Bot tab. (3) Verify your token and Groq API key are correct in the bot.py file.
Can I edit the character after downloading?+
Yes! Open bot.py in any text editor (Notepad, VS Code, etc.) and edit the character variables at the very top of the file. Or just come back here, re-fill the form, and re-download.
How do I keep the bot running 24/7?+
Use a VPS (DigitalOcean, Vultr, Railway — from $5/month) and run nohup python bot.py & or set up a systemd service. You can also use Railway's free tier for small bots.
The /generate command doesn't work — why?+
You need a valid Hugging Face token. Get one free at huggingface.co/settings/tokens. Make sure it's in the HF Token field in the generator before downloading.
What does NSFW mode actually do?+
It removes PG-13 content restrictions from the AI's responses, allowing more mature themes in conversation. Only enable this in age-verified 18+ Discord servers where it's appropriate. You are fully responsible for how it's used.
